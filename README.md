After you launch the program, you will see two panels. Each side acts as its own File Explorer. You can drag and drop items from the left side to the right side of the screen.

### Setting your favorite folders
Add folders you visit often to the address bar. Right-click the address bar and select "Add to Favorites." This saves time when you move documents between project folders.

### Adjusting the columns
You can view file details such as name, size, type, or date. Right-click the header row of any panel. Uncheck or check the categories you need. Press the "Size" column header to sort files by how much space they consume.

### Customizing colors

## 📁 Managing Folders Effectively

Professional file management depends on how you structure your data. Use the "Filter" feature at the bottom of each panel to find specific file types. Enter *.pdf into the filter box to hide every file except PDFs. This makes it easy to find specific documents in a cluttered folder.

If you deal with many compressed files, understand that the program handles Zip, 7z, and RAR files like standard folders. You can open them by pressing Enter. This prevents the need to extract files before you decide which ones to keep.
